# AI Matchmaking App

This is a simple AI-powered matchmaking app using Flask (backend) and Streamlit (frontend).

## Features
- Find compatible matches based on events.
- Random AI-generated compatibility scores.
- Simple chatbot for engagement.

## Installation
1. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```

2. Run the backend:
   ```sh
   cd backend
   python app.py
   ```

3. Run the frontend:
   ```sh
   cd frontend
   streamlit run app.py
   ```
